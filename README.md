# NODE Carousel Generator for Claude Code

Generate professional SVG carousels from any topic, powered by AI. Configure your brand once, create unlimited carousels.

## Installation

```bash
claude plugin add nodeagencyai/Carousel-plugin-claude
```

## Requirements

- An OpenRouter API key ([get one here](https://openrouter.ai))
- Set it as `OPENROUTER_API_KEY` in your environment or configure via plugin settings

## Quick Start

1. Install the plugin
2. Run `/carousel:setup` to configure your brand identity
3. Run `/carousel:generate "your topic"` to create your first carousel

## Commands

| Command | Description |
|---------|-------------|
| `/carousel:setup` | Configure your brand profile (colors, fonts, tone) |
| `/carousel:generate "topic"` | Generate a new carousel from a topic |
| `/carousel:regenerate` | Redo a specific slide with custom instructions |
| `/carousel:preview` | Open carousel preview in browser |
